How to Use

In HTML:

<!-- As an image --> <img src="devxworks-logo-highlighted-x.svg" alt="devXworks" width="200"> <!-- Inline SVG (for styling control) --> <svg width="400" height="120" viewBox="0 0 400 120"> <!-- SVG content --> </svg>

In CSS:

.logo { background-image: url('devxworks-logo-highlighted-x.svg'); background-size: contain; background-repeat: no-repeat; width: 200px; height: 60px; }

In React/JSX:

import logo from './devxworks-logo-highlighted-x.svg'; function Header() { return <img src={logo} alt="devXworks" className="logo" />; }

Converting to Other Formats

SVG to PNG: Use online converters like CloudConvert, or design tools like Figma, Adobe Illustrator

SVG to JPG: Same tools as above, but note that JPG doesn't support transparency

For Favicon: Convert the small SVG to ICO format using favicon generators

Recommended: Keep using SVG when possible - it's scalable, small file size, and perfect quality at any size
